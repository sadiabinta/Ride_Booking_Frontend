This is the frontend for the Ride-Sharing Application built with **React, Vite, Redux Toolkit, RTK Query, React Router, Tailwind CSS, and ShadCN UI**.

It supports both **Driver** and **Rider** functionalities including authentication, ride flow, location updates, earnings tracking, and more.

---

## Features

### Authentication

- Login & Registration (Rider & Driver)
- JWT token handling and secure API calls
- Auto-redirect based on role and login status

---

### Driver Features

- Go Online / Offline
- View available rides (Live)
- Accept ride request
